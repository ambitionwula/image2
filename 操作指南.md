# 造像所简易操作指南

## 1. 打开软件

- 桌面版：双击“造像所”图标。
- 开发版：浏览器访问 `http://localhost:5173`。

## 2. 首次配置接口

1. 点击页面右上角的“接口设置”。
2. 软件已默认填写以下内容，可按需修改：
   - NewAPI Base URL：`https://newapi.smartlifemarketing.com/v1`
   - 图片模型：`gpt-image-2`
   - 文案分析模型：`gpt-5.6-sol`
3. 在“API Key”中填写自己的密钥。新电脑首次打开时，此项为空。
4. 点击“测试接口”，确认接口和模型可用。
5. 设置会自动保存在当前电脑，下次打开无需重复填写。

> 请妥善保管 API Key，不要通过截图或聊天工具发送给无关人员。

## 3. 生成图片

1. 进入图片生成页面。
2. 输入希望生成的画面描述。
3. 选择图片尺寸、质量、返回格式和生成数量。
4. 点击生成按钮，等待任务完成。
5. 在结果区域预览或下载图片。

## 4. 编辑已有图片

1. 切换到图片编辑功能。
2. 上传需要修改的图片，可上传多张参考图并切换主图。
3. 输入修改要求；如果只修改局部，可使用画笔涂抹需要调整的区域。
4. 点击生成编辑结果。
5. 检查结果后下载，或继续送入编辑功能调整。

## 5. 制作电商套图

1. 进入电商套图功能并上传商品图片。
2. 填写商品信息和制作要求。
3. 选择主图、SKU 图和详情图尺寸。
4. 先进行商品分析，再根据生成的方案制作图片。
5. 任务执行时可以暂停或取消，完成后下载所需结果。

## 6. 常见问题

- 提示“请先完成接口设置”：检查 API Key 是否填写。
- 接口测试失败：检查 Base URL、API Key 和网络连接。
- 模型不可用：点击“测试接口”，从返回的可用模型中重新选择。
- 新电脑没有 API Key：这是正常的安全设计，需要在新电脑上重新填写。
- 修改过的配置如何恢复：打开“接口设置”，重新填写默认值或其他可用配置。
